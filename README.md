A 16-bit style platformer game where you play as a girl rescuing 6 adorable animals from cages, collecting color-coded keys to unlock each one. Once all animals are freed, a special Valentine's surprise awaits!

## Play

Hosted on GitHub Pages: simply open the link in your mobile browser.

## Controls

- **Mobile**: On-screen D-pad (left side) + Jump button (right side)
- **Desktop**: Arrow keys to move, Space to jump

## How to Deploy on GitHub Pages

1. Create a new GitHub repository
2. Push all files to the `main` branch
3. Go to **Settings â†’ Pages**
4. Set source to **Deploy from a branch**, select `main` / `/ (root)`
5. Your game will be live at `https://<username>.github.io/<repo-name>/`

## Tech

- Vanilla HTML5 Canvas + JavaScript
- No dependencies or build step
- All pixel art rendered programmatically
